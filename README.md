## Steps
### 1. Clonar el repositorio
```bash
$ git clone https://github.com/
```
### 2. Instalar las dependencias

```bash
$ yarn install
```

### 3. Instalar Nest CLI

```bash
$ npm i -g @nestjs/cli
```

### 4. Levantar la base de datos

```bash
$ docker-compose up -d
```
